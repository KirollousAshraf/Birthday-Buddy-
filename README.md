# Birthday App 🎂

A simple React application that displays a list of people celebrating their birthdays today.

## Features

View all birthday celebrants with their photo and age
Remove individual people from the list
Clear all birthdays at once
Refresh the list back to its original state

## Concepts Practiced

useState for state management
Props & props drilling
children prop for reusable components
Array methods (filter) for state updates
Component composition & separation of concerns

## Tech Stack

React JS
CSS3 (Grid Layout)
